# Vibrantly

## Build Tools

**Front End:** EJS, Materialize

**CRUD operations:** Express

**Back End Databases:** PostgreSQL, Firebase, Knex.js

**Middleware:** Express-Session, bCrypt, body-parser

**Project Management:** ZenHub - integrated with GitHub
